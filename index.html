Введите функцию поведения шаттла

Туториал по Марс Лэндру.
Цель игры:
Цель нашей игры заключается в написании программы, которая позволит посадить шаттл на различные поверхности во время симуляций, чтобы в конце посадить настоящий шаттл вместе с первыми колонизаторами на настоящий Марс.

Параметры игры:

Горизонтальное положение шаттла вызывается с помощью MM.getData(1), вертикальная – MM.getData(2).
Горизонтальная скорость шаттла вызывается с помощью MM.getData(3), вертикальная же - MM.getData(4).
Угол наклона шаттла - angle, его оставшееся топливо шаттла вызывается с помощью MM.getData(5) shuttle.shuttleFuel
Угол наклона шаттла – angle, является одной из двух переменных, которые можно менять в вашей программе. Ограничен значениями между -90 и 90, влияет на направление силы от двигателя.
Скорость шаттла зависит воздействия гравитации и направление силы двигателя, вызываемое “power”. Скорость ограничена значениями от 0 (бездействие) до 10 (максимальная мощь).
Шаттл меняет свою модель поведения после вызова return с аргументами [power, angle]
Гравитация планеты составляет 3.71 единица

Параметры миссии:
Геймплей игры разделен на миссии, которые проверят вашу программу на универсальность. В каждой миссии вы будете начинать из разных положений и пытаться приземлиться на разнообразные типы поверхностей. Поверхности состоят из точек, между которыми проведены линии. Расположение точек можно узнать через функцию MM.getData(6), которая возвращает список точек данной миссии. У каждой точки есть свое значение горизонтального и вертикального положения.
На каждой миссии будет своеобразное плато, на которое нужно будет посадить шаттл. Плато это та часть поверхности марса между двумя соседними точками с одинаковым вертикальным положением.
